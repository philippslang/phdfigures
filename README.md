# phdfigures
code and data to generate selected plots of my phd thesis

## Run
Requires Python 3. Optionally setup a virtual environment
```
python3 -m venv venv
source venv/bin/activate
# or with conda
conda create -n phdfigures python=3.6
activate phdfigures
```
to run
```
pip install -r requirements
python generate.py
```
